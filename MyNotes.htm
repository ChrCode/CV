HTML Lists

The HTML Lists can be customized with the "type" attribute to select the bullet for the items in the list. For Consistency the type should be defined at the list level rather than at the list element.

Unordered List <ul>

The Unordered List should be used when the order of the items in the list doesn't matter

Each Elements in the list need to be identified as an <li> (list element) element

The synstax is:
<h3>HTML List Types</h3>
<ul>
<li>Unordered List (ul)</li>
<li>Ordered List (ol)</li>
<li>Description List (dl)</li>
...
<li type="1">Error</li>
</ul>

And would apprears like this:

HTML List Types

  • Unordered List <ul>
  • Ordered List <ol>
    • ...
  • Description List <dl>
  • Error

In an Unordered List the bullet can be changed using the Type Attribute

Ordered List <ul>

The Ordered List should be used when the order of the items in the list is important. Like a top ten list, or a list of topics to be covered in a given order.

Each Elements in the list need to be identified as an <li> (list element) element

The synstax is:
<h3>Ordered List Numbering Types</h3>
<ol>
<li type="a"> Type "a" Indicates lowercase letters</li>
<li type="A"> Type "A" Indicates uppercase letters</li>
<li type="i"> Type "i" Indicates lowercase roman numerals</li>
<li type="I"> Type "I" Indicates uppercase roman numerals</li>
<li type="1"> Type "1" Indicates numbers (This is the default)</li>
...
<li type="Square">Error</li>
</ol>

Ordered List Numbering Types

  1. Type "a" Indicates lowercase letters
  2. Type "A" Indicates uppercase letters
  3. Type "i" Indicates lowercase roman numerals
  4. Type "I" Indicates uppercase roman numerals
  5. Type "1" Indicates numbers (This is the default)
    6. ...
  6. Error

In an Unordered List the bullet can be changed using the Type Attribute

Description List <dl>

Common uses for this element are to implement a glossary or to display metadata (a list of key-value pairs).

Each Elements in the list need to have a <dt> Description Term and an <dl> Description List elements

HTML List Types

Unordered List
Typically rendered as a bulleted list

The element encloses a list of groups of terms
